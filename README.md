# ☕ Brew Haven Cafe - Landing Page

Welcome to the **Brew Haven Cafe** repository! This project is a modern, cozy coffee shop landing page built completely from scratch as my very first step into web development.

---

## 🎨 Current Features (Hero Section)
* **Warm & Cohesive Palette**: Thoughtfully selected beige and espresso tones that evoke a relaxing cafe aesthetic.
* **Sleek Interactive Buttons**: Custom rounded call-to-action buttons styled with modern CSS transition and hover effects.
* **Optimized Image Layout**: Fully responsive coffee cup hero image with customized smooth border-radii.

## 🛠️ Tech Stack
* **HTML5**: For semantic structuring of the hero section, textual content, and buttons.
* **CSS3**: For layout design, typography alignment, and utilizing **Flexbox** for clean horizontal/vertical element distribution.

## 📈 Key Learnings
Building this section helped me fundamentally master several essential core front-end concepts:
1. Deconstructing the crucial functional difference between internal and external spacing (`Padding` vs. `Margin`).
2. Shifting and aligning layout components cleanly using modular CSS property hierarchies.
3. Resetting default browser margins and utilizing CSS specificity rule overriding (CSS Cascade).
4. Structuring child wrapper containers (`div` tags) to prepare layouts for fluid responsive structures.
